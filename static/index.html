Deadabase API Reference

Deadabase is a REST API for browsing and exploring Grateful Dead concert history. Search through thousands of shows by date, venue, song, or set name - with full setlists and show footnotes.

Base URL: https://deadabase.onrender.com

Replace the example localhost:8080 with the base URL above unless you clone the GitHub repo and are running the API locally.

All endpoints return JSON and respond to GET requests only.
Songs that segue into each other within a set are represented as a single string joined with >. footnotes is keyed by marker when present, empty object otherwise.

GET /shows

Returns a filtered list of shows. Note: this is a summary shape - no sets included. Fetch the individual show by its ID (GET /shows/{id}) for the full setlist.

Parameter Type Description Example
start_date / end_date string Shows within a date range start_date=1970-01-01&end_date=1970-01-31
song string Shows where this song was played song=althea
set_name string Shows containing a given set (e.g. acoustic, encore) set_name=acoustic
venue string Shows at a venue matching this name venue=fillmore
has_notes bool Shows that have notes attached has_notes=true has_notes=false
year int Shows from a given year year=1973
location string Shows in a given state or country. States should be standard 2 letter abbreviations. Case-insensitive location=NY location=england

Songs

GET /songs/{name}

Returns stats for a single song.

Example

curl 'http://localhost:8080/songs/althea'

Response

{
    "times_played": 271,
    "first_played": "1979-08-04",
    "last_played": "1995-07-08"
}

GET /songs

Returns a filtered/sorted list of songs. Query parameters split into filters (narrow the result set) and directives (change how results are sorted).

Parameter Kind Description Example
sort directive Sort order for results (e.g. most_played) sort=most_played
set_name filter Restrict to songs played within a given set (set_1, set_2, set_3, encore, acoustic, electric) set_name=encore
venue filter Songs played at a given venue venue=dream
played_lt filter Songs played fewer than N times played_lt=30
